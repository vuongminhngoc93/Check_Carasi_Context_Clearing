SYSTEM REQUIREMENTS - Check Carasi DF Context Clearing Tool
=============================================================

MINIMUM REQUIREMENTS:
--------------------
Operating System:
- Windows 10 (64-bit) Build 1903 or later
- Windows 11 (any version)
- Windows Server 2019/2022

Framework & Runtime:
- .NET Framework 4.7.2 (REQUIRED)
- Microsoft Visual C++ 2015-2022 Redistributable (x64)

Microsoft Office:
- Microsoft Office 2016, 2019, 2021, or Microsoft 365
- Excel component (for .xls/.xlsx file processing)
- Outlook component (for email features - optional)

Database Engine:
- Microsoft Access Database Engine 2016 Redistributable (x64)
- Required for OLEDB Excel file reading

Hardware:
- RAM: 4GB minimum, 8GB recommended
- CPU: Dual-core 2.0GHz minimum
- Disk Space: 500MB free space
- Display: 1366x768 minimum resolution

PERFORMANCE OPTIMIZATIONS:
-------------------------
With Option 6 Hybrid Approach:
- Single Variable Search: 1-2 seconds
- Batch Search (50+ variables): 10x performance improvement
- Cache Hit Rate: 80-90% for repeated searches
- Memory Usage: Optimized for 30-50 tabs
- Connection Pooling: Max 10 concurrent OLEDB connections

NETWORK REQUIREMENTS:
--------------------
- No internet connection required for core functionality
- Local network access for file sharing (if applicable)
- SMTP access for email features (optional)

DEPENDENCIES TO INSTALL:
-----------------------
1. .NET Framework 4.7.2
   Download: https://dotnet.microsoft.com/download/dotnet-framework/net472
   
2. Microsoft Access Database Engine 2016 Redistributable (x64)
   Download: https://www.microsoft.com/download/details.aspx?id=54920
   
3. Visual C++ 2015-2022 Redistributable (x64)
   Download: https://aka.ms/vs/17/release/vc_redist.x64.exe

OPTIONAL COMPONENTS:
-------------------
- Microsoft Office suite (for advanced Excel features)
- Microsoft Outlook (for DD request email functionality)

TESTED ENVIRONMENTS:
-------------------
- Windows 10 Pro (Build 19044) + Office 2019 ✅
- Windows 11 Enterprise + Microsoft 365 ✅  
- Windows Server 2022 + Office 2021 ✅

KNOWN LIMITATIONS:
-----------------
- Maximum 60 tabs (Windows Forms TabControl limit)
- Large Excel files (>100MB) may impact performance
- COM Interop requires local Office installation for some features
